# Customer Segmentation Analysis Dashboard (RFM Analysis)

## Project Overview
This project focuses on Customer Segmentation Analysis using the RFM (Recency, Frequency, Monetary) model in Excel and Power Query.

The objective of the analysis was to identify high-value customers, understand customer purchasing behaviour, and provide business recommendations based on customer activity and spending patterns.

## Tools Used
- Microsoft Excel
- Power Query
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization Techniques

## Dataset Information
The dataset contains transactional retail data with the following fields:
- Invoice Number
- Stock Code
- Description
- Quantity
- Invoice Date
- Unit Price
- Customer ID
- Country

## Analysis Performed
The following RFM metrics were calculated:

### Recency (R)
Measures how recently a customer made a purchase.

### Frequency (F)
Measures how often a customer purchases products.

### Monetary (M)
Measures the total amount spent by each customer.

Customers were segmented into:
- Champions
- Loyal Customers
- Regular Customers
- At-Risk Customers
- Big Spenders

## Dashboard Features
The dashboard includes:
- KPI Cards
- Customer Segment Distribution
- Revenue by Segment
- Top 10 Customers
- Interactive Slicers

## Key Insights
- The analysis reveals thst Champions represent the largest customer segment, indicating a strong case of highly engaged and high-value customers.
- Loyal customers form the second largest segment, showing a significant number of customers consistently engsge with the business over time.
- The big spenders segment contain the fewest customers, indicating that although high-value purchases exist, they are concentrated aming a limited number of customers.

## Recommendations

- Continue strengthening relationships with Champions through exclusive offers, loyality rewards and personalized engagement strategies to maintain customer satissfaction and retention.
- Loyal customerrs should be nurtured through targeted promotions and consistent customer experience initiatives to encourage progression into the Champions segment.
- Special Premium packages and exclusive deals can be introduced to encourage higher spending among existing customers and expand the Big Spenders segment.
